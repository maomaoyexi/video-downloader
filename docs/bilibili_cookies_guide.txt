============================================================
         Bilibili 1080p 下载 Cookie 配置指南
         对应版本：v1.9.5 WebUI 多平台版
============================================================

文件存放：cookies.txt 需与主程序放在同一目录
配置方式：程序默认使用 cookies.txt 文件模式

------------------------------------------------------------
一、为什么 Bilibili 1080p 需要 Cookie？
------------------------------------------------------------

Bilibili 从 2024 年底起调整了视频 API 策略：

- 360P/480P：无需登录，直接可下载
- 720P：部分情况需要登录
- 1080P 高清（标准）：需要登录，但无需大会员
- 1080P 高码率：需要大会员
- 4K 超清：需要大会员

关键机制：Bilibili API 根据 Cookie 中的 CURRENT_QUALITY 值
决定返回哪个画质的视频流。因此导出 Cookie 时的画质设置
直接影响下载时可获取的最高画质。

------------------------------------------------------------
二、CURRENT_QUALITY Cookie 详解
------------------------------------------------------------

这是 Bilibili 下载 1080p 最关键的技术细节：

| CURRENT_QUALITY | 对应画质       | 登录要求 | 大会员 |
|-----------------|---------------|---------|--------|
| 120             | 4K 超清       | 是      | 是     |
| 112             | 1080P 高码率   | 是      | 是     |
| 80              | 1080P 高清     | 是      | 否     |
| 64              | 720P 高清     | 部分    | 否     |
| 32              | 480P 清晰     | 否      | 否     |
| 16              | 360P 流畅     | 否      | 否     |

Bilibili 每次请求只返回一个 premium 画质格式！
因此，导出 Cookie 前必须先在浏览器中切换到目标画质。

------------------------------------------------------------
三、推荐配置方法（三步搞定 1080p）
------------------------------------------------------------

方法一：浏览器扩展导出（推荐，最稳定）

1. 安装扩展
   - Chrome/Edge：搜索 "Get cookies.txt LOCALLY"
   - Firefox：搜索 "cookies.txt"

2. 切换到 1080p 画质（关键步骤！）
   - 在浏览器中打开任意 Bilibili 视频
   - 点击播放器右下角的「画质」按钮
   - 选择「1080P 高清」（不是 1080P 高码率）
   - 等待视频以 1080p 开始播放

3. 导出 Cookie
   - 立即点击扩展图标
   - 选择「Export」导出为 Netscape 格式
   - 保存到本地

4. 放置文件
   - 将导出的文件重命名为：cookies.txt
   - 放到工具目录下（与主程序同级）

方法二：浏览器自动提取（更方便，推荐日常使用）

1. 在浏览器中登录 Bilibili
2. 切换到 1080p 画质播放视频
3. 在工具的「设置」页面中：
   - Cookie 模式选择「浏览器提取」
   - 选择你使用的浏览器（Chrome/Edge/Firefox）
   - 保存设置
4. 工具会自动从浏览器提取 Cookie，无需手动导出

------------------------------------------------------------
四、手动验证 CURRENT_QUALITY
------------------------------------------------------------

打开导出的 cookies.txt，搜索 CURRENT_QUALITY，确认值为 80：

  .bilibili.com  TRUE  /  FALSE  1765432100  CURRENT_QUALITY  80

如果值是 0 或 64，说明导出时画质未设为 1080p，需重新操作。

------------------------------------------------------------
五、常见问题排查
------------------------------------------------------------

1. 问题：提示 "Format(s) 1080P 高清 are missing"
   原因：CURRENT_QUALITY Cookie 值不是 80
   解决：
   - 在浏览器中先切换到 1080p 画质播放视频
   - 立即重新导出 Cookie
   - 确认 cookies.txt 中 CURRENT_QUALITY=80

2. 问题：1080p 下载成功但只有 720p
   原因：账号未登录，或 Cookie 已失效
   解决：重新登录 Bilibili，确认在 www.bilibili.com 已登录

3. 问题：想下载 1080P 高码率或 4K
   前提：需要大会员账号
   步骤：
   - 在浏览器中切换到目标画质（1080P高码率 或 4K）
   - 立即导出 Cookie
   - CURRENT_QUALITY 应为 112 或 120

4. 问题：Cookie 过期频繁
   Bilibili Cookie 有效期较短（通常 1-7 天），建议使用
   「浏览器提取」模式自动获取最新 Cookie。

5. 问题：下载链接过期 (0 bytes read)
   原因：视频直链有时效性（deadline参数），获取后需
   在有效期内完成下载
   解决：重新开始下载即可

6. 问题：HTTP 429 速率限制
   Bilibili 有严格的请求频率限制
   解决：降低并发线程数（建议 Bilibili 使用 2-3 线程），
   工具和 yt-dlp 会自动处理退避重试

------------------------------------------------------------
六、快速检查清单
------------------------------------------------------------

□ 已在浏览器中登录 bilibili.com
□ 已在播放器中切换到 1080P 高清画质
□ 在切换画质后立即导出了 Cookie
□ cookies.txt 中 CURRENT_QUALITY 值为 80
□ 文件与主程序放在同一目录
□ Cookie 文件为 Netscape 格式
□ 账号非大会员时，不要选择 1080P 高码率或 4K

------------------------------------------------------------
七、安全提示
------------------------------------------------------------

1. cookies.txt 包含登录凭证，请勿分享
2. 定期清理旧 Cookie 文件
3. 使用公共电脑后记得删除
4. 截图反馈问题前检查是否暴露了 Cookie 内容
